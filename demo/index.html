üí° Critical: CSS Import for Styling

Important: The most critical part of using this package is importing the CSS file first!

The showcase demonstrates the proper import pattern:

// Import CSS FIRST for proper styling import '@avinash_tyagi/design-system-react/lib/index.css'; // THEN import components import { Button, Card, Header } from '@avinash_tyagi/design-system-react';

Without the CSS import: Components render as plain HTML without styling
With the CSS import: Components render with proper colors, spacing, and interactive effects

üîß Troubleshooting

Components appear unstyled?

Make sure you import the CSS file first in your main app file:

import '@avinash_tyagi/design-system-react/lib/index.css';

Import errors?

Verify the package is installed correctly:

npm list @avinash_tyagi/design-system-react

Missing components?

Check that you're importing from the correct package name and the component exists in the latest version.
